The first two options require a Google OAuth Client ID from Google Cloud Console.
The third option skips all of that.

You need a free Google Cloud project to issue OAuth credentials. One-time setup.

  1. Go to Google Cloud Console and create a project
  2. Navigate to APIs & Services → Library
  3. Search Google Sheets API and click Enable
  4. Also enable Google Drive API (needed for receipt photos)
  1. In Cloud Console → APIs & Services → Credentials
  2. Click Create Credentials → OAuth 2.0 Client ID
  3. Application type: Web application
  4. Under Authorized JavaScript origins add:
  5. Click Create and copy the Client ID

Not a secret — only works from origins you authorised in Cloud Console.
